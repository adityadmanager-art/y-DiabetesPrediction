# Diabetes Prediction — Project Overview

A machine learning project that predicts the onset of diabetes in patients using the **PIMA Indians Diabetes Dataset** sourced from Kaggle. Multiple classification algorithms and neural network architectures are explored and compared.

---

## 🔍 Purpose

Early detection of diabetes can significantly improve patient outcomes. This project builds and evaluates several predictive models to classify whether a patient is likely to have diabetes based on diagnostic measurements.

---

## ✨ Features

- **Multiple ML algorithms** evaluated side by side (SVM, MLP, ELM, Dense Neural Network, DeepForest)
- **High accuracy** — Dense Layer model achieves up to 100% accuracy at 170 epochs
- **Fast inference** — prediction completed in ~14 seconds after model training
- **Clean dataset** — uses the well-known PIMA Diabetes CSV dataset
- **Jupyter Notebooks** for interactive exploration and experimentation
