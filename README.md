# 🩺 Diabetes Prediction — Machine Learning Project

> Predicting the onset of diabetes using the PIMA Indians Diabetes Dataset with multiple machine learning algorithms and neural network architectures.

---

## 📌 About

This repository contains a collection of machine learning models built to predict whether a patient is diabetic, based on diagnostic health measurements. It is designed as both a practical ML experiment and a learning resource.

**Dataset:** PIMA Indians Diabetes Dataset (sourced from Kaggle)  
**Task:** Binary Classification — Diabetic (1) or Non-Diabetic (0)  
**Models Implemented:** SVM, MLP, ELM, Dense Neural Network, DeepForest
